# Fibras Musculares Tipo II (Blancas / Rápidas)

> **Fuente:** Brusco & Belingeri, *Histología Médico-práctica* (2da ed.), Cap. 6, p.113-114

![[imgs/01_esqueletico/26_tabla-fibras-tipo-I-IIA-IIB.jpg]]
*Tabla comparativa de tipos de fibras musculares*

## 📋 Características

| Propiedad | Descripción |
|-----------|-------------|
| **Color** | Blanco |
| **Nombre alternativo** | Rápidas, Type II, Glucolíticas |
| **Tamaño** | Grandes |
| **Mioglobina** | Bajo contenido → color blanco |
| **Mitocondrias** | Menos abundantes |
| **Capilares** | Menos numerosas |
| **Sistema energético** | **Glucolítico** (anaeróbico) |
| **Resistencia a la fatiga** | **Baja** |
| **Velocidad de contracción** | Rápida |
| **Potencia** | Alta |

## 🔬 Características ultraestructurales
- Menor contenido de mioglobina
- Menos mitocondrias que las rojas
- Menor vascularización
- Abundante glucógeno (reserva para glucólisis anaeróbica)

## 🏃 Función
- Contracción **rápida y potente**
- **Baja tolerancia a la fatiga**
- Ideales para movimientos **breves pero precisos** y de alta potencia

## 📍 Localización y ejemplos
- Músculos que controlan el movimiento de los **dedos**
- Músculos **extrínsecos del ojo** (movimientos oculares rápidos y precisos)
- Músculos de miembros superiores

## 📊 Subtipos de fibras (según isoforma de miosina)
- **Fibras de contracción rápida, glucolíticas (FG):** glucólisis anaeróbica, fatiga rápida
- **Fibras de contracción rápida, oxidativas (FO):** oxidadas + glucólisis, tolerancia intermedia

> 💡 **Catedra:** la proporción de fibras blancas aumenta en los miembros superiores (movimientos breves y precisos).
